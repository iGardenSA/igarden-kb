## 📚 مفاهيم تقنية مهمة

### Token = الوحدة الأساسية

```
Token ≈ 4 أحرف إنجليزية ≈ 0.75 كلمة
أو
Token ≈ 1.5-2 حرف عربي ≈ 0.5 كلمة عربية
```

⚠️ **العربية تستهلك tokens أكثر من الإنجليزية بـ 2-3×.**

### Context Window = "ذاكرة قصيرة المدى"

```
200,000 tokens إجمالي:
├── System Prompt          ~10,000 tokens
├── Project Knowledge      ~50,000 tokens
├── Tool Definitions       ~20,000 tokens
└── Available for chat     ~120,000 tokens
```

### Compaction = ضغط المحادثات

عند امتلاء Context Window:
1. Claude يأخذ المحادثة الكاملة
2. يُولّد ملخصاً (~5K tokens)
3. يستبدل المحادثة بالملخص
4. **التفاصيل الدقيقة تضيع**

⚠️ **هذه المشكلة ظهرت في هذه الجلسة بالذات.**

## 🛠️ Claude Code — الأداة الأهم

### ما هو:
أداة سطر أوامر تشغّل Claude **داخل terminal** على جهازك. يستطيع:
- قراءة ملفاتك مباشرة
- تنفيذ أوامر bash
- تعديل الكود
- git commit + push

### تكلفة:
- API tokens (Sonnet 4.5 افتراضياً)
- ~$1-3 لكل جلسة عمل عادية

### الاستخدام في iGarden:
- ✅ كل البرمجة على Pi 5
- ✅ إصلاح أخطاء Vercel
- ✅ تحديث Knowledge Base (مستقبلاً)
- ✅ أتمتة المهام التقنية

### من الجوال:
**لا يوجد Claude Code من الجوال مباشرة.** لكن يمكن:

#### الحل: SSH من الجوال إلى Pi/CM5

```
جوالك (Termius)
        │
        │ SSH عبر WiFi/4G
        ▼
   Pi 5 أو CM5 (في عسفان)
        │
        ├── claude code (مثبّت محلياً)
        ├── git, npm, node
        └── كل شيء
```

**التطبيقات المقترحة:**
- **Termius** (Android + iOS) — مجاني
- **JuiceSSH** (Android) — ممتاز
- **Blink Shell** (iOS) — احترافي

#### الحل المتقدّم: Tailscale + SSH

```
جوالك في أي مكان (حتى 4G)
        │
        ▼
   Tailscale VPN (مجاني)
        │
        ▼
   Pi 5 (في عسفان أو الأحساء)
```

**الفائدة:** تتصل بالـ Pi من أي مكان كأنك في نفس الشبكة.

## 🎯 قواعد ذهبية للاستخدام الاحترافي

### 1. اختر النموذج حسب المهمة
- **القرار قبل التنفيذ:** Opus
- **التنفيذ نفسه:** Sonnet
- **الأتمتة المتكررة:** Haiku

### 2. ابدأ محادثات منفصلة للمواضيع المنفصلة
**لا تخلط:**
- مراجعة عقود + برمجة + Pitch deck في محادثة واحدة
- كل موضوع له context مختلف، Claude يفقد التركيز

### 3. استثمر في Project Knowledge
كل محادثة جديدة تستفيد من ملفات Project Knowledge المحدّثة.

### 4. كن مُحدّداً في طلباتك
```
❌ ضعيف: "ساعدني في عميل الأحساء"
✅ قوي: "أريد BOM للـ 8 zones بميزانية محدّدة"
```

### 5. استخدم Artifacts للملفات الكبيرة
بدلاً من نسخ كود طويل في المحادثة.

### 6. تعلّم Prompt Engineering
- Be specific
- Use examples
- Specify format
- Set constraints
