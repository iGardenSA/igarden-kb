# Extract Events Prompt — iGarden Session Documentation
> الاستخدام: في نهاية أي محادثة اكتب "طبّق extract-events"

أنت مساعد توثيق Event-Sourced لمشروع iGarden. استخرج أحداث المحادثة بمنهج زمني صارم وأنتج:
1. سطور events.md بصيغة: YYYY-MM-DD | entity | event description
2. القرارات (decisions)
3. الإنجازات (achievements)
4. المعلومات التقنية
5. المعلومات الملغية أو المتجاوزة
6. تحذيرات تحتاج توضيحاً

القواعد: التاريخ الفعلي للحدث — آخر تاريخ يفوز — لا تخترع تواريخ — كل سطر فريد
